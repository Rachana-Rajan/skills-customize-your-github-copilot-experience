# 📘 Assignment: Statistics with Python

## 🎯 Objective

Use `pandas` and `numpy` to calculate statistical metrics and interpret data distributions. Students will load a dataset, compute summary statistics, and visualize results through code.

## 📝 Tasks

### 🛠️ Load and explore dataset

#### Description
Read a CSV dataset with pandas and generate initial exploratory data output.

#### Requirements
Completed program should:

- Load `data.csv` using `pd.read_csv()`
- Display first five rows and dataset shape
- Provide column data types and missing value counts

### 🛠️ Compute key statistical metrics

#### Description
Calculate central tendency and dispersion metrics using pandas and numpy.

#### Requirements
Completed program should:

- Compute mean, median, mode for a numeric column
- Compute standard deviation and variance
- Compute 25th, 50th, 75th percentiles
- Print results in a readable format

### 🛠️ Visualize data category and distribution

#### Description
Create simple plots for distribution and category frequency.

#### Requirements
Completed program should:

- Plot histogram for a numeric column
- Plot bar chart for a categorical column count
- Save plots as `histogram.png` and `bar_chart.png`

## 💡 Optional extensions

- Add inline comments describing each statistic’s meaning
- Calculate correlation matrix and highlight strong relationships
- Add a function to generate summary reports automatically
